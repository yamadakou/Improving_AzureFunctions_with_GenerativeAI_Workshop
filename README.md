# 「生成AI × Azure Functions ワークショップ」用リポジトリ

## ワークショップ概要
生成AI使った設計改善・レビュー・テストコードへの活用を体験する2時間のワークショップです。
実際に ChatGPT や GitHub Copilot を使ってダメコードを改善しながら、生成AIを体験してみましょう！

## 事前準備
事前にVisual Studio CodeのインストールとGitHubアカウント（会社のメールアドレスでもOKです）を用意し、 Visual Studio Code で「GitHub Copilot」を利用できるようにしてください。

### 「GitHub」のURL
https://github.com/

### 「Visual Studio Code」のURL
https://azure.microsoft.com/ja-jp/products/visual-studio-code
※社内PCの場合、「Y:\AppList\Microsoft\VSCode」にインストーラが格納されています。

### GitHub Copilot を自分用に設定する
https://docs.github.com/ja/copilot/how-tos/set-up/set-up-for-self

### 環境への GitHub Copilot 拡張機能のインストール
https://docs.github.com/ja/copilot/how-tos/set-up/install-copilot-extension
